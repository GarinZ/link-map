## 💡 Link Map是什么？
Link Map是一个浏览器标签侧边栏，更适用于重度信息消费者。
在Link Map中即使你有大量的Tab和Window，仍然可以通过树形视图进行归类和整理保持清晰的逻辑性，再进一步的构建自己的链接收藏夹

## ✨ 主要功能
- 🌲 树形垂直标签侧边栏：更多层级并且可以迅速通过拖拽完成更多的Tab操作
- 💾 关闭并保存：清空工作区并聚焦于当下的工作
- 🔒 备份和恢复：支持导出为`.json`和恢复
- 📶 无需联网：所有内容都存在本地，保证自己的信息安全
